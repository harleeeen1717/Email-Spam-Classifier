# Email Spam Classifier
## Internship ID CITS5136
## Project Overview
This project is a Machine Learning-based Email Spam Classifier that predicts whether a message is Spam or Not Spam.

The model uses Natural Language Processing (NLP) techniques to convert text messages into numerical features and applies the Multinomial Naive Bayes algorithm for classification.

## Objective
To develop a machine learning model that can automatically detect unwanted spam messages and separate them from normal messages.

## Technologies Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- CountVectorizer
- Multinomial Naive Bayes
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- Multinomial Naive Bayes

## Dataset
SMS Spam Collection Dataset

The dataset contains SMS messages labeled as:
- Ham (Not Spam)
- Spam

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction using CountVectorizer
5. Model Training
6. Model Testing
7. Spam Prediction

## Model Result
The trained model successfully classifies messages as Spam or Not Spam with good accuracy.

## How to Run
1. Open the Jupyter Notebook/Google Colab file.
2. Install required libraries.
3. Run all cells sequentially.
4. Enter a message to check whether it is Spam or Not Spam.
